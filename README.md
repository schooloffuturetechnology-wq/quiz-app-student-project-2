# Quiz App

A polished static quiz application built with plain HTML, CSS, and JavaScript. It is ready for GitHub hosting through the repository root.

## Features

- responsive single-page quiz experience
- question navigation map with progress tracking
- previous and next navigation without losing answers
- final score, accuracy, and answer review
- static setup ready for GitHub Pages

## Files

- `index.html` - hosting entry point
- `style.css` - application styling
- `app.js` - quiz logic and rendering

## Run locally

Open `index.html` in a browser.

## GitHub hosting

Use the repository root as the GitHub Pages source so `index.html` is served directly.
